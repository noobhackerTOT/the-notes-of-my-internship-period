# Markdown 语法速览

Markdown 语法速览

Markdown 是一种轻量级标记语言，让你用纯文本格式就能编写可读性高、易于转换为 HTML 的文档。以下是最常用的语法：

### 1. 标题

使用 `#`​ 号，数量表示级别，**后面必须有空格**。

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 2. 强调

- **加粗**：`**加粗文字**`​ 或 `__加粗文字__`
- *斜体*：`*斜体文字*`​ 或 `_斜体文字_`
- ***粗斜体***：`***粗斜体***`
- ~~删除线~~：`~~删除线~~`

### 3. 列表

**无序列表**：使用 `-`​、`+`​ 或 `*` 加空格

```markdown
- 项目 1
- 项目 2
  - 嵌套项目（缩进 2 或 4 空格）
```

**有序列表**：数字加英文句点加空格

```markdown
1. 第一项
2. 第二项
   1. 嵌套有序（缩进 2-4 空格）
```

**任务列表**（GitHub 风格）：

```markdown
- [x] 已完成项
- [ ] 未完成项
```

### 4. 链接

```markdown
[显示文本](https://example.com "可选标题")
<https://直接链接.com>        <!-- 自动链接 -->
[带引用的链接][id]            <!-- 引用式 -->
[id]: https://example.com    <!-- 引用定义在文末 -->
```

### 5. 图片

```markdown
![替代文本](图片URL "可选标题")
![本地图片](/path/to/image.png)
```

### 6. 代码

**行内代码**：用反引号包裹 `` `代码` ``​  
**代码块**：三个反引号或四个空格缩进，可指定语言

```markdown
‍```python
def hello():
    print("Hello, world!")
‍```
```

### 7. 引用

```markdown
> 这是一段引用。
> 可以跨行。
>> 嵌套引用。
```

### 8. 表格（GitHub 风格）

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 单元格1 | 单元格2  | 单元格3 |
| 长内容  | 居中     |    右   |
```

- 冒号控制对齐：`:---`​ 左对齐，`:---:`​ 居中，`---:` 右对齐

### 9. 分割线

单独一行使用三个或以上的 `-`​、`*`​ 或 `_`：

```markdown
---
***
___
```

### 10. 转义字符

使用反斜杠 `\`​ 输出 Markdown 中的特殊字符，如 `\*`​ 输出 `*`​，`\#`​ 输出 `#`。

### 11. HTML 支持

大部分 Markdown 解析器允许直接嵌入 HTML 标签，如 `<div>`​、`<span>`​、`<img>` 等。

### 12. 扩展语法（依实现而异）

- **任务列表**（见上）
- **脚注**：`[^1]`​，文末写 `[^1]: 脚注内容`
- **定义列表**：`术语\n: 定义`
- **自动链接**：`[user@example.com](mailto:user@example.com)`

### 💡 提示

- 不同的 Markdown 解析器（GitHub、Typora、思源笔记等）在细节上可能有差异，例如表格、脚注、高亮等的支持程度。
- 遇到不生效的语法，先检查**空格**、**空行**以及所用平台支持的扩展功能。

掌握以上语法，你就能写出清晰、漂亮的 Markdown 文档了！
