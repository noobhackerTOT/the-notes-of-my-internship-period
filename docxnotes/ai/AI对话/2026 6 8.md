# **一、网络基础篇**

|协议|用途|面试关注点|
| -------------| --------------------| ----------------------------------------------------------------------|
|TCP|面向连接，可靠传输|三次握手、TIME\_WAIT、防止重传重复包|
|UDP|无连接，不可靠|DNS、DHCP常用，快速但不保证顺序|
|DNS|域名解析|查询流程：本地 → 域名服务器 → 根服务器；DNS Tunnel可能用于隐蔽通信|
|HTTP|应用层明文|抓包可直接看到请求内容|
|HTTPS / TLS|加密传输|TLS握手协商密钥、SNI泄漏域名、证书可信性|
|NTP|时间同步|日志分析依赖准确时间，防止事件错位|

# **二、Windows安全日志及含义**

spn：SPN 是 **Service Principal Name（服务主体名称）**

|日志ID|含义|阶段/作用|SOC关注点|
| --------| ------------------| -------------------------| --------------------------------------------------|
|4624|登录成功|初始访问|Type、来源IP、账户是否异常|
|4625|登录失败|初始访问|高频失败可能密码喷洒 / 暴力破解|
|4688|新进程创建|执行阶段|父子进程、命令行（PowerShell参数最重要）、C2启动|
|4769|Kerberos TGS请求|横向移动 / 服务票据收集|请求SPN、频率、加密类型（RC4注意）|
|7045|新服务安装|持久化 / 提权|服务名称、路径、启动账户、创建时间|

# **三、典型攻击链（ATT&CK映射）**

1. **初始访问**

   - 邮件钓鱼 / 恶意文档
   - 日志：4624/4625
   - ATT&CK：T1204（User Execution）
2. **执行**

   - PowerShell / CMD / Office宏
   - 日志：4688
   - ATT&CK：T1059.001（PowerShell）
3. **横向移动**

   - SMB / RDP / WMI
   - 日志：4624 Type3/10, 4688
   - ATT&CK：T1021.x（Remote Services）
4. **权限提升 / 服务票据收集**

   - Kerberoasting → 离线破解服务账户密码
   - 日志：4769
   - ATT&CK：T1558.003
5. **持久化 / 提权**

   - 安装服务 / 注册表 / gMSA
   - 日志：7045 / 4688
   - ATT&CK：T1050 / T1547
6. **C2通信 / 数据窃取**

   - HTTP / HTTPS / DNS Tunnel
   - 日志：网络流量监控
   - ATT&CK：T1071.001（Web Protocols）

---

# **四、SOC重点关注点**

1. **登录行为**

   - 4624/4625：异常账号、来源IP、登录类型（Type3/10）
2. **进程行为**

   - 4688：父子进程链、PowerShell命令行参数、可疑可执行文件
3. **Kerberos票据**

   - 4769：请求SPN、频率、加密类型（RC4）、请求来源主机
4. **持久化 / 提权行为**

   - 7045：新服务名称、路径、启动账户、创建时间
5. **网络流量异常**

   - DNS请求频率异常（DNS Tunnel）
   - HTTP/HTTPS频繁访问未知域
   - 外部IP C2通信
6. **攻击链关联分析**

   - 登录 → 执行 → 横向移动 → 提权/票据收集 → 持久化 → C2

# **五、防御 & 建议措施**

|攻击行为|防御措施|SOC监控|
| --------------------------| -----------------------------| --------------------------------|
|密码喷洒 / Kerberoasting|强密码策略 / gMSA / 审计SPN|4769日志，关注RC4、频繁TGS请求|
|恶意PowerShell / CMD|AppLocker / AMSI / 禁用宏|4688日志，父子进程链分析|
|横向移动|MFA / 网络分段 / 限制RDP|4624 Type3/10异常登录|
|持久化服务安装|最小权限 / 白名单|7045日志分析|
|DNS隧道|DNS请求模式监控|关注异常子域、数据量、频率|
